# Danh sách hóa đơn

**Danh sách hóa đơn** là nơi lưu trữ tập trung toàn bộ các hóa đơn đã được hệ thống xử lý thành công, bao gồm hóa đơn tải về từ NCC/TCT và hóa đơn đã qua bước tra cứu, xác minh. Tại đây, bạn có thể dễ dàng quản lý, tìm kiếm và trích xuất dữ liệu.

1. Vào menu **Danh sách hóa đơn**.
2. Sử dụng bộ lọc nguồn dữ liệu hoặc các điều kiện tìm kiếm nếu cần.
3. Bấm **Lọc** để tìm kiếm hoặc **Xóa lọc** để đưa điều kiện về mặc định.
4. Bấm **Xuất Excel** nếu cần xuất dữ liệu phục vụ đối soát/báo cáo.
