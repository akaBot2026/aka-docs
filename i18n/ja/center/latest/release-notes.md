# Akabot Center — リリースノート

## v3.2.0 — 2026年4月

### 新機能
- **Autopilotオーケストレーション** — トリガー条件と目標を自然言語で記述すると、Centerがスケジュールとキュー構成を自動生成します。
- **ロボットヘルスダッシュボード** — 接続されている各ロボットのCPU、メモリ、セッション状態をリアルタイムで表示する専用ビューを追加。
- **環境プロモーションウィザード** — 承認ゲートを含むDev → UAT → Prodへのプロセスパッケージ昇格を案内するワークフロー。

### 改善
- キュー項目の優先度が3段階から10段階に拡張され、より細かなディスパッチ制御が可能になりました。
- 実行ログ検索が正規表現パターンと日付範囲フィルタをサポートしました。
- `/processes` および `/jobs` エンドポイントのAPI応答時間が約35%改善されました。

### バグ修正
- 同一秒内に2つのスケジュールが発火した際にジョブが重複してディスパッチされる競合状態を修正しました。
- GMT+7タイムゾーンのロボットで誤った「最終確認」タイムスタンプが表示される問題を修正しました。
- ユーザーでフィルタした場合の監査ログのページネーション不具合を修正しました。

## v3.1.0 — 2026年1月

### 新機能
- **マルチテナンシー** — 単一のCenterインストールで複数の分離されたテナント（ユーザー、ロボット、プロセス）をホスト可能になりました。
- **Webhookトリガー** — 外部システムからのHTTP POSTでプロセスを起動できます。
- **SLAトラッキング** — キューごとに想定完了時間を設定し、SLA違反時にアラートを出します。

### 改善
- ロール管理UIを権限グループテンプレート（Operator, Developer, Admin）で再設計しました。
- パッケージアップロードサイズ制限を50MBから200MBに拡張しました。

### バグ修正
- スケジュール無効化時に次の保留トリガーが即座に停止しない問題を修正しました。
- 項目が放棄され再試行された際のキュースループット指標を修正しました。

## v3.0.0 — 2025年9月

### 新機能
- **Center 3.0 ポータル** — Reactベースでアクセシビリティ（WCAG 2.1 AA）を改善した完全再設計されたWeb UI。
- **高可用性モード** — Raftコンセンサスを用いた自動リーダー選出を含むアクティブ-パッシブフェイルオーバー。
- **詳細な監査ログ** — すべてのAPIコール、ログイン、設定変更がIPおよびユーザーエージェント付きで記録されます。

### 破壊的変更
- v2 REST API (`/api/v2/`)は非推奨です。v4.0リリース前に`/api/v3/`へ移行してください。
- v2.xからのロボットマシンキーはアップグレード後に再登録が必要です。

### バグ修正
- 高負荷時のキュー処理でのデータベースデッドロックを解消しました。
- ディレクトリサーバーが部分結果を返した際にLDAPグループ同期がサイレントに失敗する問題を修正しました。

## v4.0.0.5 — 2025年4月29日

### 主な強化点

- **追加**: Agent Poolの有効/無効機能。

- **更新**: 日本語サイドメニューの翻訳改善（Integration Service, Data Service, Automation Insights）。
- **修正**: 複数画面でのページネーションとフィルタリング不具合。
- **修正**: ジョブ重複状態の不一致。

2. スケジューリング & トリガー改善
- **追加**: ファンクショントリガーでのAPIコールアクティビティサポート。
- **追加**: スケジュール/トリガーの条件ロジック。
- **追加**: スケジュールジョブ重複クリーンアップ機構。
- **追加**: SchedulerからAgent Poolへのフォワーディングロジック。
- **修正**: 関連アイテムが存在する場合にスケジュールが削除できない問題(SQL Server)。
- **修正**: Cron式の高度ケースにおけるバリデーション。

3. アプリ統合 & API強化
- **追加**: application integration APIでのspecificData & correctionIdサポート。
- **更新**: Application Integration UIのソートロジック改善。

4. Agent Pool強化
- **修正**: Agent Poolの優先度問題。
- **修正**: Agent Poolから作成されたキュー項目参照の上書き問題。

5. ライセンス管理強化
- **更新**: ライセンス画面の改良。
  - Deactivateボタンがライセンスごとに操作可能になりました。
  - LMSライセンス取得ボタンを追加しました。

6. セキュリティ & フレームワーク更新
- **更新**: quartz-schedulerバージョンアップおよび依存関係の整理。
- **更新**: Google Guavaの更新（TPIセキュリティ要件）。
- **更新**: Spring Boot 3.3.0 → 3.3.6にアップグレード。
- **更新**: Angular 14 → 17にアップグレード。
- **更新**: ShareUIをv3.0へアップグレード。
- **修正**: 複数のセキュリティ関連検証と依存性の不具合修正。

7. データベース & システム互換性
- **追加**: PostgreSQLサポート。
- **更新**: Hibernate検索の再インデックス/リフレッシュ機能。
- **要件**: Tomcat 10
- **要件**: JDK 17
