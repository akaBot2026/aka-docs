# IT運用チーム向け

## **ユースケース1：組織ユニット（OU）の作成 / 編集**

S1: 左メニューの**Administration**をクリックし、**Organization Unit**タブを選択して組織ユニットページにアクセスします。

![image-20230425154231-1.png](/static/img/8c5b48_image-20230425154231-1.png)

S2: OUの詳細を表示するには、**目のアイコン（view）**をクリックします。

![image-20230425154231-2.png](/static/img/f81b09_image-20230425154231-2.png)

詳細ページが表示されます。

![image-20230425154231-3.png](/static/img/292f96_image-20230425154231-3.png)

|  |  |
| --- | --- |
| カラム/ラベル | 説明 |
| Name | OUの名前 |
| User | このOUに割り当てられているすべてのユーザーの一覧（ユーザー名、メール等） |

S3: OUを作成/編集するには

**Create New**ボタンをクリックします。既存OUを編集する場合は、該当OU名の横にあるActionからEditを選択します。

![image-20230425154231-4.png](/static/img/e2261a_image-20230425154231-4.png)

S4: クリックすると入力用のポップアップフォームが表示されます。

![image-20230425154231-5.png](/static/img/f9b39f_image-20230425154231-5.png)

|  |  |
| --- | --- |
| カラム/ラベル | 説明 |
| Name | 作成/編集するOUの名前を入力します |
| Description | OUの説明を入力します |
| User | OUに追加するユーザーを選択します（複数追加可能だが一人ずつ追加） |

## **ユースケース2：OUの削除**

**Organization Unit**メニューで、ページ上の対象OU名の横にある**Delete**ボタンをクリックするとOUを削除できます。

***注: OUを削除すると、そのユニット内のすべてのリソースが削除されます***

![image-20230425154231-6.png](/static/img/030999_image-20230425154231-6.png)

## **ユースケース3：ロールの作成/編集**

ロールは権限の集合であり、特定のCenterリソースへのアクセスや操作に必要な権限をロールに割り当てます。

S1: メニューで**Administration**を選択し、**Role Management**タブを選んで各ロールの権限を確認します。

![image-20230425154231-7.png](/static/img/d44c49_image-20230425154231-7.png)

S2: ページ上部の**Create New**ボタンをクリックして新しいロールを作成します。

![image-20230425154231-8.png](/static/img/44618b_image-20230425154231-8.png)

S3: ポップアップフォームが表示されるので入力します。

![image-20230425154231-9.png](/static/img/71cfa0_image-20230425154231-9.png)

S4: 権限にチェックを付けてロールに権限を付与し、**Save**をクリックします。

|  |  |
| --- | --- |
| **Permissions** | **カテゴリー** |
| None | ユーザーはCenterのUIでリソースを表示できません |
| View | ユーザーはリソースを表示できますが操作はできません |
| Edit | ユーザーはリソースを表示および編集できます |
| Create | ユーザーはリソースを表示および新規作成できます |
| Delete | ユーザーはリソースを削除できます |

S5: 既存のロールを編集するには、Action列の該当ロールの**Edit**ボタンをクリックします。

![image-20230425154231-10.png](/static/img/5d3552_image-20230425154231-10.png)

## **ユースケース4：ロールの削除**

S1: ロールを削除するには、該当ロールの**Delete**ボタンをクリックします。

![image-20230425154231-11.png](/static/img/1b3b1e_image-20230425154231-11.png)

S2: 確認ポップアップが表示されるので**Delete**をクリックして操作を完了します。

![image-20230425154231-12.png](/static/img/c836cf_image-20230425154231-12.png)

## **ユースケース5：アカウントの表示/編集/作成**

S1: 左メニューで**Administration > Users**タブを選択すると、以下のような詳細ページが表示されます。

![image-20230425154231-13.png](/static/img/c775ef_image-20230425154231-13.png)

S2: ユーザーを表示するには目のアイコンをクリックします。

![image-20230425154231-14.png](/static/img/fe1abe_image-20230425154231-14.png)

画面は以下のように表示されます。

![image-20230425154231-15.png](/static/img/39d7f9_image-20230425154231-15.png)

|  |  |
| --- | --- |
| **カラム** | **説明** |
| Status | ユーザーの状態（Activated または Deactivated） |
| Profiles | ユーザーに割り当てられたロール。デフォルトで3つのロールがあります: ROLE_ADMIN, ROLE_USER, ROLE_ROBOT。ユーザーは複数のロールを持つことができます。各ロールの権限はRole Managementセクションで説明されています |

S3: アカウントを作成/編集するには、右側のCreate Newをクリックします。

![image-20230425154231-16.png](/static/img/1ee1eb_image-20230425154231-16.png)

**Create New/Edit**ボタンをクリックすると、入力フォームが表示されます。詳細は下表を参照してください。

![image-20230425154231-17.png](/static/img/6a800f_image-20230425154231-17.png)

フォームに入力後、**Save**をクリックして保存します。作成を取り消す場合は**Cancel**をクリックします。

## **ユースケース6：アカウントの削除**

ユーザーを削除するには、該当ユーザーの**Delete**ボタンをクリックします。

![image-20230425154231-18.png](/static/img/7e5f3e_image-20230425154231-18.png)

## **ユースケース7：ホームページでのセンター概要の確認**

S1: 左メニューでHomepageを選択します。このページには以下の情報が表示されます：

- エージェント数（Available、Busy、Disconnected、Nonresponse）
- タスク数（Successful、Running、Stopped、Faulted）
- スケジュール（時間、日、週、月単位で計算）
- パッケージ（Active、Inactive）
- タスク統計（Daily、Weekly、Monthly、Yearlyの表示）
- 今後のタスク
- 最近のタスク

S2: **Filter**ボタンでAgent GroupやAgent Nameでタスクを絞り込めます。

![image-20230425154231-19.png](/static/img/3e7b8e_image-20230425154231-19.png)

S3: タスクを素早く作成するには、この**“Plus”**ボタンをクリックします。

![image-20230425154231-20.png](/static/img/455256_image-20230425154231-20.png)

## **ユースケース10：エージェントの状態監視**

S1: 左メニューのResourceをクリックし、Agentタブを選択するとエージェントの詳細が確認できます。

![image-20230425154754-27.png](/static/img/f53bee_image-20230425154754-27.png)

|  |  |
| --- | --- |
| **カラム/ラベル** | **説明** |
| **Agent Group** | エージェントが所属するエージェントグループ |

| **Version** | ローカルマシンにインストールされているakaBot Agentのバージョン |
| **Created by** | エージェントを作成したユーザー |

## **ユースケース15：ボットの実行 / 停止**

1. 手動でボットを実行する（タスク作成、クローン）

**オプション1:**

S1: 左メニューでAutomationをクリックし、**Task**タブを選択します。

![image-20230425155109-46.png](/static/img/2e05ac_image-20230425155109-46.png)

S2: 手動で実行したいワークフローを選択して**保存**します。

![image-20230425155109-47.png](/static/img/d5c239_image-20230425155109-47.png)

**オプション2:** タスク行のActionをクリックして表示されるドロップダウンで**Clone**を選択し、**Save**をクリックします。

![image-20230425155109-48.png](/static/img/1f0ac2_image-20230425155109-48.png)

2. スケジュールによるボット実行（スケジュール設定、スケジュール無効化）

**2.1. スケジュールの作成 / 編集:**

S1: メニューで**Automation**を選び、**Schedule**タブをクリックして**Create New**を押します。

![image-20230425155109-49.png](/static/img/72b022_image-20230425155109-49.png)

S2: ポップアップフォームに入力します。

|  |  |
| --- | --- |
|
* **Name (*):** スケジュール名
* **Workflow (*):** スケジュール対象のワークフローを選択
* **Time Zone (*):** スケジュールで使用するタイムゾーンを選択
**Triggerタブ:**
* **Recurrence:** スケジュールの周期を選択：Minutes/ Hourly/ Daily/ Weekly/ Monthly/ Advance
* **Start Date (*):** スケジュールの開始日時を選択
* **End Date:** スケジュールの終了日時を選択
**Execution Targetタブ:** 選択したワークフローに応じて適切なエージェントを選択
**Parametersタブ:**
**Holiday settingsタブ:** 単一選択
* Run continuously (デフォルト)
* Bypass holiday
* Postpone until the next workday
***注: (*)は必須項目です***

![image-20230425155109-50.png](/static/img/image-20230425155109-50.png)

選択肢が**Once**の場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| At | スケジュールの開始時刻を指定 |

**Minutes**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| End Date | スケジュールの終了日を選択 |
| Every | 分単位での周期を指定 |

**Hourly**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| End Date | スケジュールの終了日を選択 |
| Every … Hours | 時間単位での周期を指定 |

**Daily**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| End Date | スケジュールの終了日を選択 |
| At | スケジュールの開始時刻を指定 |

**Weekly**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| End Date | スケジュールの終了日を選択 |
| Everyday At | スケジュールの開始時刻を指定 |
| Monday Tuesday Wednesday Thursday Friday Saturday Sunday | 時間・分・曜日で周期を指定 |

**Monthly**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Start Date | スケジュールの開始日を選択 |
| End Date | スケジュールの終了日を選択 |
| At | スケジュールの開始時刻を指定 |
| On | 月内の特定の日を指定 |
| Of | 年内の特定の月を選択 |

**Advance**を選択した場合

|  |  |
| --- | --- |
| **カラムラベル** | **説明** |
| Cron Expression | 空白で区切られた6または7フィールドの文字列で、カレンダーに基づく再帰的なスケジュールを作成できます |

2.2. **スケジュールの開始 / 停止**

B1: スケジュールを強制的に開始/停止するには、Actionボタンをクリックして**Start Now**または実行中なら**Stop**を選択します。

![image-20230425155109-51.png](/static/img/acd650_image-20230425155109-51.png)

2.3. スケジュールの無効化 / 有効化:

S1: スケジュールを無効化するには、該当のActionボタンをクリックし**Disable**を選択します。

![image-20230425155109-52.png](/static/img/a28347_image-20230425155109-52.png)

S2: クリック後、状態切替の確認ダイアログが表示されます。

![image-20230425155109-53.png](/static/img/2ad1e3_image-20230425155109-53.png)

スケジュールが無効化されると、そのステータスは**Disable**に変わります。

S3: スケジュールを有効化するには、該当のActionボタンをクリックし**Enable**を選択します。

![image-20230425155109-54.png](/static/img/81bca4_image-20230425155109-54.png)

2.4. **スケジュールの削除:**

B1: 削除するには、該当スケジュールのActionボタンをクリックし**Delete**を選択します。

![image-20230425155109-55.png](/static/img/4110bb_image-20230425155109-55.png)

3. ボットの停止

S1: 実行中のボットを停止するには、**Task**タブで該当タスク行のActionをクリックし、表示されるドロップダウンで**Stop**を選択します。

![image-20230425155109-56.png](/static/img/0b6352_image-20230425155109-56.png)

S2: 確認メッセージが表示されます。**Confirm**を押してタスクを削除します。

![image-20230425155109-57.png](/static/img/2a65ce_image-20230425155109-57.png)
