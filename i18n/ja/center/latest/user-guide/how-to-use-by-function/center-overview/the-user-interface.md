akaBot Centerのユーザーインターフェースは、明快さと運用効率を重視したWebアプリケーションです。レイアウトは、常に表示される左側のナビゲーションサイドバー、上部のヘッダーバー、選択したモジュールに応じて変化するメインコンテンツ領域で構成されています。この一貫した構造により、実行中ジョブの監視、レポートの確認、エージェント管理など異なる操作機能を文脈を失うことなく迅速に切り替えられます。

上部のヘッダーバーには現在のユーザー名とプロファイルオプション、システム通知、クイックアクセス設定が表示されます。通知ベルアイコンは、ジョブの失敗、エージェントの切断、ライセンスの有効期限警告などの重要イベントをリアルタイムで通知します。通知をクリックすると該当する詳細ページへ直接移動でき、問題の診断や対応時間を短縮します。

メインコンテンツ領域はアクティブなモジュールに応じてレイアウトが変わります。リストビューはソート可能かつフィルタ可能なテーブルを使用してジョブ、エージェント、パッケージなどのレコードを表示します。詳細ビューでは設定フォーム、実行ログ、ステータスインジケータを表示します。Homeのようなダッシュボードビューはウィジェットベースのレイアウトでチャートやカウンタを使い、一目で運用状況を把握できます。

## UIレイアウト概要

| No | パネル | 説明 |
| --- | --- | --- |
| 1 | **機能メニュー** | akaBot Centerの主要機能を表示 |
| 2 | **クイックナビゲーション** | ページ、パッケージ、エージェント、エージェントグループ、ワークフロー、アセット、キュー、スケジュールなどへ素早く移動 |
| 3 | **メイン画面** | 各機能の詳細を表示 |
| 4 | **クイック作成** | タスク/スケジュール/ワークフロー/トリガー/エージェントを素早く作成 |
| 5 | **組織ユニット選択メニュー** | アクティブな組織ユニットを切り替え |
| 6 | **言語** | 表示言語を変更（英語、韓国語、日本語、ベトナム語、繁体字） |
| 7 | **ユーザーメニュー** | **Settings**（氏名、メール、表示言語の変更） **Password**（ログインパスワードの変更） **Sign out**（サインアウト） |
| 8 | **バージョン** | akaBot Centerのバージョン表示 |
