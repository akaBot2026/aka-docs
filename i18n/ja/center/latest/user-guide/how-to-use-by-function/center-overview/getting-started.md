akaBot Centerへようこそ — RPA運用の集中管理プラットフォームです。本ガイドでは、システムにアクセスして構成を開始し、チームが自動化ワークフローのデプロイと管理を開始するための初期手順を説明します。続行する前に、akaBot Centerがインストールされアクセス可能であり、システム管理者から有効なログイン資格情報を受け取っていることを確認してください。

サポートされるウェブブラウザを開き、akaBot CenterインスタンスのURL（例：`http://your-server:8080`）にアクセスしてログインします。初回ログイン時にはデフォルトパスワードの変更を求められることがあります。認証後、Centerのホームダッシュボードが表示され、実行中ジョブ、保留中のスケジュール、エージェントの稼働状況などの概要が確認できます。

ログイン後の推奨セットアップ順序は、まずエージェントを登録し、次にグループまたはプールに整理し、次に自動化パッケージをアップロードし、最後にワークフローとスケジュールを構成することです。この順序により、オーケストレーションを開始する前に実行リソースが揃っていることが保証されます。

## 初期セットアップチェックリスト

- [ ] akaBot CenterのWebインターフェースにアクセスしてログインする
- [ ] デフォルトパスワードを変更する（初回ログイン時）
- [ ] 少なくとも1台のakaBotエージェントを登録する
- [ ] エージェントグループまたはエージェントプールを作成する（任意、推奨）
- [ ] Studio経由で自動化パッケージをアップロードする
- [ ] パッケージをエージェントまたはプールに紐づけるワークフローを作成する
- [ ] 実行を自動化するためのスケジュールまたはトリガーを設定する
- [ ] ホームダッシュボードでジョブ実行を確認する

## Centerにアクセスするためのシステム要件

| Requirement | Recommended |
|---|---|
| Browser | Chrome 90+, Edge 90+, Firefox 88+ |
| Network | Centerサーバーへのアクセス（デフォルトポート：8080） |
| Account | 最低でもViewerロールを持つ有効なユーザーアカウント |
| Screen Resolution | 1280 x 720以上 |
