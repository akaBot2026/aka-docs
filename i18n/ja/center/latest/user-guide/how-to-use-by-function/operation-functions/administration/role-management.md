# ロール管理

akaBotはロールと権限に基づくアクセス制御フレームワークを採用しています。ロールは権限の集合であり、特定のCenterリソースへのアクセスや操作に必要な権限をロールに割り当てます。ユーザーは個別にリソース権限を直接付与されるのではなく、ロールを通じて権限を取得します。ロールに応じて、ユーザーは以下のようなリソースへアクセスできます：

| No | リソース |
| --- | --- |
| 1 | 管理（ADMIN）リソース |
| 2 | 環境（ENVIRONMENT）リソース |
| 3 | ロボット（ROBOT）リソース |
| 4 | パッケージ（PACKAGE）リソース |
| 5 | ワークフロー（WORKFLOW）リソース |
| 6 | ジョブ（JOB）リソース |
| 7 | スケジュール（SCHEDULE）リソース |
| 8 | 組織ユニット（ORGANIZATION UNIT）リソース |
| 9 | アセット（ASSET）リソース |
| 10 | キュー（QUEUE）リソース |
| 11 | ログ（LOG）リソース |

| 権限 | カテゴリー |
| --- | --- |
| None | ユーザーはCenterのUIで該当リソースを表示できません |
| View | ユーザーはリソースを表示できますが操作はできません |
| Edit | ユーザーはリソースを表示および編集できます |
| Create | ユーザーはリソースを表示および新規作成できます |
| Delete | ユーザーはリソースを削除できます |

このRole Managementページでは利用可能なすべてのロールが表示され、管理者はロールの追加、削除、情報更新を行えます。**Role Management**タブを選択すると次のページが表示されます：

![image-20221101164359-16.png](/static/img/d407b3_image-20221101164359-16.png)

| No | カラム | 説明 |
| --- | --- | --- |
| 1 | Action | ユーザー管理に利用可能なアクション（Edit: ロールの詳細表示、Delete: ロールの削除） |
| 2 | Name | ロール名 |

* デフォルトロール

akaBot Centerにはデフォルトで以下の3つのロールがあります：

- ROLE_ADMIN
- ROLE_USER
- ROLE_ROBOT

**ROLE_ADMIN**  
デフォルトで全リソースに対するすべての権限を持ち、変更できません。

**ROLE_USER**  
デフォルトで以下の権限を持ちます（編集可能）。

![image-20230704162245-1.png](/static/img/27201e_image-20230704162245-1.png)

**ROLE_ROBOT**  
デフォルトで以下の権限を持ちます（編集可能）。

## a. ロールの作成/編集

新しいロールを作成するには、ページ上部の**Create New**ボタンをクリックします。

![image-20221101164549-17.png](/static/img/aacffe_image-20221101164549-17.png)

既存ロールを編集するには、Action列の該当ロールの**Edit**ボタンをクリックします。

![image-20221101164638-18.png](/static/img/c7556f_image-20221101164638-18.png)

**Create New/Edit**ボタンをクリックすると入力フォームが表示されます。詳細は以下の通りです：

![image-20230704162458-3.png](/static/img/96e8d2_image-20230704162458-3.png)

| No | カラム/ラベル | 説明 | タイプ | 最大長 | 必須 | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | Name | ロール名を入力 | String | 100文字 | はい | |
| 2 | Resources | 各行について1つの権限を選択してロールの権限を設定 | SingleChoice |  | はい | |

## b. ロールの削除

ロールを削除するには、該当ロールの**Delete**ボタンをクリックします。

![image-20221101164723-19.png](/static/img/f253ab_image-20221101164723-19.png)

確認ポップアップが表示されます。**Delete**を押して操作を完了してください。
