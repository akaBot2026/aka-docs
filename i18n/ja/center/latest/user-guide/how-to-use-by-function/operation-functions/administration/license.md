## ライセンス

エージェント/StudioライセンスはCenterのLicenseページで組織ユニット単位に管理できます。これにより、デプロイの効率化やライセンス有効化を一元管理できます。ボタン操作でライセンスの有効化、更新、無効化が可能で、年間のライセンス利用状況も追跡できます。

**License Management**タブを選択すると、以下のページが表示されます：

![image-20221101170138-29.png](/static/img/29f2d4_image-20221101170138-29.png)

- ライセンスの追加

https://docs-v2.akabot.com/center/latest/installation/license-activation

- ライセンスの更新

https://docs-v2.akabot.com/center/latest/installation/license-renewal
