# ライセンス

Agent/Studio ライセンスは、センターのライセンス ページに基づいて組織単位でアクセスできます。これにより、展開を加速し、単一の場所からアクティベーションを管理できるようになります。ボタンを押すだけで、ライセンスをアクティブ化、更新、またはキャンセルできます。年間を通じてライセンスの使用状況を追跡することもできます。

[**ライセンス管理**] タブを選択すると、次のページが表示されます。
