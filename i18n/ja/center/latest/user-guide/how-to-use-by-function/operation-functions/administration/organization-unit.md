# 組織ユニット

**Organization Units（OU）**は、オーケストレーターのコンポーネントを指定ユーザーごとにテナント分離するための仕組みです。ユーザーがCenterで閲覧・操作できる範囲は、所属する組織ユニットと割り当てられたロールによって決まります。組織ユニットには以下のエンティティが含まれます：

- エージェントグループ
- エージェント
- ワークフロー
- タスク
- スケジュール
- パッケージリポジトリ
- アセット
- キュー
- ログ
- レポート

各組織ユニットごとに統計が作成され、ダッシュボードに表示されます。

**注意:**
- 組織ユニットが割り当てられていないユーザーはリソースにアクセスできません。
- 同一エージェントを異なる組織ユニットにプロビジョニングすることはできません。リソースを別の組織ユニットへ移動することはできません。
- ページヘッダーにはユーザーに割り当てられたすべてのユニットがドロップダウンで表示され、いつでもOUを切り替えることができます。

![image-20221101164936-20.png](/static/img/1d8887_image-20221101164936-20.png)

OUページにアクセスするには、左メニューのAdministrationをクリックしOrganization Unitタブを選択します。作成済みのOUが一覧表示されます。

![image-20221101165208-21.png](/static/img/682836_image-20221101165208-21.png)

管理者として、特定OUの詳細表示、作成、更新、削除、フィルタリング、およびそのOUに割り当てられたユーザーの管理が行えます。

## a. OUの表示

OUの詳細を表示するには、**目のアイコン（View）**をクリックします。

クリックすると詳細ページに遷移します。

![image-20221101165318-22.png](/static/img/445096_image-20221101165318-22.png)

| No | カラム/ラベル | 説明 |
| --- | --- | --- |
|  | **Name** | OUの名前 |
|  | **Description** | OUの説明 |
|  | **User** | このOUに割り当てられたユーザー一覧（ユーザー名、メール、割り当てロール） |

## b. OUの作成/編集

新しい組織ユニットを作成するには**Create New**ボタンをクリックします。既存OUを編集するには、該当OU名の横のActionからEditを選択します。

![image-20221101165443-23.png](/static/img/654459_image-20221101165443-23.png)

クリックすると入力フォームのポップアップが表示されます。

![image-20221101165552-24.png](/static/img/acd773_image-20221101165552-24.png)

| No | カラム/ラベル | 説明 | タイプ | 最大長 | 必須 | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | **Name** | 作成/編集するOUの名称を入力 | String | 255文字 | はい | |
| 2 | **Description** | OUの説明を入力 | String | 1000文字 | いいえ | |
| 3 | **User** | OUに追加するユーザーを選択 | 検索入力 | 制限なし | はい | 複数追加可能だが一度に1人ずつ追加する |

ユーザーをOUから外すにはEdit画面で該当ユーザーのゴミ箱アイコンをクリックします。すべて入力/編集後、**Save**をクリックしてOUを作成/更新します。

## c. OUの削除

**重要！** 組織ユニットを削除すると、そのユニット内のすべてのリソースが削除されます。組織ユニットを削除するには、Organization Unitページで該当ユニット名の横にある**Delete**ボタンをクリックします。

![image-20221101165636-25.png](/static/img/d1ec82_image-20221101165636-25.png)

確認ポップアップが表示されます。**Delete**を選択して操作を完了してください。
