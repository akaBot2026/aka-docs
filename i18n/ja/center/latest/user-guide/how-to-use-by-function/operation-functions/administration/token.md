## トークン

Tokenページでは、サードパーティにアカウント/パスワードを渡すことなくAPIにアクセスするためのトークンを簡単に作成できます。

ダッシュボードのAdministrationタブからTokenタブを選択すると、作成済みトークンの一覧が表示されます。

![image-20221101170819-30.png](/static/img/284198_image-20221101170819-30.png)

| No | カラム/ラベル | 説明 |
| --- | --- | --- |
| 1 | Action | トークン管理に利用可能なアクション（Edit: トークン編集、Delete: トークン削除） |
| 2 | Alias Name | トークンを送信するサードパーティの名称 |
| 3 | Name | トークンの名称 |
| 4 | Description | トークンの説明 |
| 5 | Roles | トークンへアクセス可能なロール |
