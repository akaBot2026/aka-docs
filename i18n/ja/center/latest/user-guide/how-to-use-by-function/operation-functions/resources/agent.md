# エージェント

エージェントは、akaBot Studioで作成されたプロセスを実行する実行ホストです。

エージェントページでは、エージェントの追加・編集・状態確認、ランタイム設定の変更、割り当てられているエージェントグループの管理が行えます。

**注意:** エージェントの状態がBUSY（プロセス実行中）の場合、Username、Machine、Typeの編集はできません。エージェントページへは左メニューの「Agent」タブをクリックしてアクセスします。クリックすると、既存のエージェント一覧が表示されます。

| No | カラム/ラベル | 説明 |
| --- | --- | --- |
| 1 | **Agent Group** | エージェントが所属するエージェントグループ |
| 2 | **Name** | エージェント名（クリックで詳細表示） |

## b. エージェントのタスク表示

詳細ページ下部には、そのエージェントが実行したすべてのタスク一覧が表示されます。

![image-20221031154931-5.png](/static/img/61d3b2_image-20221031154931-5.png)

タスクに関する詳細はTaskセクションで説明します。

## c. エージェントのログ表示

詳細ページで**Task**タブ横の**Log**タブをクリックするとエージェントのログを表示できます。

![image-20221031155216-6.png](/static/img/6634c9_image-20221031155216-6.png)

**Export Log**をクリックすると、すべてのログまたはフィルタ済みログをエクスポートできます。

![image-20221031155405-7.png](/static/img/ee2568_image-20221031155405-7.png)

* 接続検証（Verify）

Centerとエージェントが有効なユーザー名とパスワードで正しく接続されているか確認するには、**Verify**ボタンをクリックします。

![image-20221031155517-8.png](/static/img/d5e373_image-20221031155517-8.png)

接続中にこのボタンを押すと、ID/Passwordの認証チェックが実行され、正しくログイン可能かが判定されます。

検証タスクの状態は次のいずれかです：

- **PENDING**: 検証タスクが作成され、エージェントの検証待ち
- **SUCCESSFUL**: 検証成功（ユーザー名／パスワードが正しい）
- **FAULTED**: 検証失敗（ユーザー名／パスワードが正しくない）

検証タスクとVerifyボタンはリアルタイムで更新されます。検証タスクが保留中の場合、Verifyボタンは無効になります。Verifyボタンを押すと検証タブが自動的に開き、右上に「Verifying...」の通知が表示されます。

作成されたすべての検証リクエストを表示するには、**Agentタブ > View agent detail > Verification**タブを選択します。

![image-20221031155727-9.png](/static/img/076928_image-20221031155727-9.png)

## d. エージェントの作成/更新

エージェントをCenterに接続する最初の手順は、Center上でエージェントを作成することです。

**注意**: ステータスがBUSYのエージェントは編集できません。新しいエージェントを作成するには、画面右上の**Create New**ボタンをクリックします。

![image-20221031155823-10.png](/static/img/474aef_image-20221031155823-10.png)

エージェントを編集するには一覧のEditボタン、または詳細ページのEditをクリックします。

![image-20221031155905-11.png](/static/img/5dd967_image-20221031155905-11.png)

クリックすると、エージェント作成/編集用のプロパティ入力ポップアップが表示されます。必要事項を入力してください。

![image-20221031155928-12.png](/static/img/941015_image-20221031155928-12.png)

エージェントタブ

| No | カラム/ラベル | 説明 | タイプ | 最大長 | 必須 | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | Key | エージェントの識別子。ローカルAgentのAgent Keyフィールドに入力します。 | String | 50文字 | はい | システムが自動生成します |
| 2 | Name | エージェント名を入力 | String | 50文字 | はい | |
| 3 | Machine Name | エージェントがデプロイされているコンピュータ名を入力 | String | 50文字 | はい | |
| 4 | Machine Username | 上記コンピュータのログインユーザー名を入力 | String | 50文字 | いいえ | |
| 5 | Machine Password | 上記コンピュータのログインパスワードを入力 | String | 255文字 | いいえ | |
| 6 | Type | エージェントのタイプを選択（DEVELOPMENT、ASSISTANT、STAGING、PRODUCTION） | ドロップダウン（単一選択） |  | はい | |
| 7 | Description | エージェントの説明を入力 | String | 500文字 | いいえ | |

エージェント設定タブ

| No | カラム/ラベル | 説明 | タイプ | 最大長 | 必須 | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | Console / RDP | akaBotはインタラクティブなWindowsセッションでワークフローを実行します。Robotが開始できるセッションはConsoleまたはRDPの2種類です。 | 単一選択 |  | はい | |
| 2 | Login To Console | RDPを選択した場合に利用可能。Consoleセッションへ接続するかどうかを指定します。 | チェックボックス |  | いいえ | |
| 3 | Font Smoothing | テキスト認識を向上させます（RDP選択時のみ利用可）。 | チェックボックス |  | 任意 | |
| 4 | Resolution Width / Height / Depth | 自動化プロセスでカスタム解像度が必要な場合に設定します（RDP選択時のみ）。 | 整数 |  | 任意 | 0より大きい値 |
| 5 | Others | その他エージェント固有のパラメータ | String | 制限なし | 任意 | |

フォーム入力後、**Save**をクリックしてエージェントを登録します。Agent Keyを使用してローカルAgent側で接続手順に従い、接続を完了してください。

* エージェントの検索

必要なエージェントを検索するには、検索ボックスにエージェント名、またはエージェントグループ・ステータスで絞り込みを行います。

![image-20221031160353-13.png](/static/img/b24682_image-20221031160353-13.png)

## e. エージェントの削除

削除するには、**Delete**ボタンをクリックします。

![image-20221031160620-14.png](/static/img/6be193_image-20221031160620-14.png)

または各エージェントの先頭のチェックボックスを選択すると、Create Newの横に**Delete**ボタンが表示されます。Action列のチェックボックスで一覧のすべてを選択して一括削除できます。

![image-20221031160701-15.png](/static/img/f5ef98_image-20221031160701-15.png)
