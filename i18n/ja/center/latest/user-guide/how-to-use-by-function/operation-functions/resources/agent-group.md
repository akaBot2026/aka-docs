# エージェントグループ

エージェントグループはプロセスをデプロイするためにエージェントをまとめたグループです。**Agent Group**ページでは既存のグループが一覧表示され、グループ内のエージェントを管理できます。名前や説明で検索・フィルタリングが可能です。

![image-20221031161241-16.png](/static/img/9a4345_image-20221031161241-16.png)

| No | カラム/ラベル | 説明 |
| --- | --- | --- |
| 1 | **Action** | エージェントグループ管理のアクション（Edit: 詳細編集、Delete: グループ削除）。各エージェントのチェックボックスを選択すると、フィルタ横に削除オプションが表示され、一括削除が可能です。 |
| 2 | **Name** | エージェントグループ名（クリックで詳細表示）。 |
| 3 | **Description** | エージェントグループの説明 |
| 4 | **Created by** | グループを作成したユーザー名 |

エージェントグループの詳細を表示するには、目のアイコンをクリックします。

![image-20221031161320-17.png](/static/img/d8243b_image-20221031161320-17.png)

| No | カラム/ラベル | 説明 |
| --- | --- | --- |
| 1 | **Name** | エージェントグループ名（クリックで詳細表示） |
| 2 | **Description** | エージェントグループの説明 |
| 3 | **Created by** | 作成者 |
| 4 | **Agent** | このセクションには、グループに割り当てられたすべてのエージェントが一覧表示されます。各エージェントのバージョン、ステータス、マシン名、マシンユーザー名、タイプ、説明、作成者などを確認できます。 |

## a. エージェントグループの作成/編集

新しいエージェントグループを作成するには、画面右上の**Create New**ボタンをクリックします。

![image-20221031161529-18.png](/static/img/1baa08_image-20221031161529-18.png)

エージェントグループ情報を編集するには、Editボタンをクリックします。

![image-20221031161612-19.png](/static/img/65f66d_image-20221031161612-19.png)

Create New/Editをクリックすると入力フォームが表示されます。詳細は以下の通りです。

![image-20221031161704-20.png](/static/img/fed62b_image-20221031161704-20.png)

| No | カラム/ラベル | 説明 | タイプ | 最大長 | 必須 | 入力要件 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | **Name** | エージェントグループ名を入力 | String | 100文字 | はい | |
| 2 | **Description** | エージェントグループの説明を入力 | String | 255文字 | いいえ | |
| 3 | **Agent** | グループに追加するエージェントを選択します。エージェント横のゴミ箱アイコンで削除可能。 | 検索入力 | 制限なし | はい | 複数追加可能だが一度に1つずつ追加 |

すべて入力後、**Save**をクリックします。

## b. エージェントグループの削除

エージェントグループを削除するには、一覧の該当グループの**Delete**ボタンをクリックします。

![image-20221031162101-21.png](/static/img/c06301_image-20221031162101-21.png)

または各グループのチェックボックスを選択すると、フィルタ横に**Delete**オプションが表示されます。Action列のチェックで一覧のすべてを選択して一括削除できます。
