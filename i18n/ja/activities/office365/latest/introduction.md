# RCA.Activities.Office365

RCA.Activities.Office365 は、Microsoft Graph を介して Microsoft Office 365（OneDrive や SharePoint のファイル/フォルダーを含む）を操作するためのアクティビティを提供します。Excel ワークブック作成や SharePoint リスト操作も含まれます。

## アクティビティ

* Office365 アプリケーション スコープ
* ファイル/フォルダーのコピー
* フォルダーの作成
* ファイル/フォルダーの削除
* ファイルのダウンロード
* ファイルを PDF としてエクスポート
* ファイルとフォルダーを検索
* ファイル/フォルダーの取得
* ファイル/フォルダーの移動
* ファイル/フォルダーの共有
* ファイルのアップロード
* ファイルのチェックイン
* ファイルのチェックアウト
* ワークブックの作成
* リスト項目の追加
* リスト項目の削除
* リスト項目の更新
* リスト情報の取得
* リスト項目の取得
* 各リストに対する繰り返し
* 各リスト項目に対する繰り返し
