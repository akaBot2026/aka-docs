# リスト項目の更新

RCA.Activities.Office365.UpdateListItem

## **説明**

指定したリスト項目を更新します。

![update-list-item.png](/static/img/update-list-item.png)

(*必須項目の場合あり)

## **アクティビティの本文内**

* **List Item** - 更新対象のリスト項目。

## **プロパティ**

**入力**

* **List Item: InArgument<Office365SharepointListItem>** - 更新対象のリスト項目。

**その他**

* **Public (Checkbox)** - アクティビティを公開する場合はチェックしてください。使用前にデータセキュリティ要件を考慮してください。

* **Display Name (String)** - このアクティビティの表示名。コードの整理や構造化のためにアクティビティ名を編集できます。
  例: [3424325] Open Window
