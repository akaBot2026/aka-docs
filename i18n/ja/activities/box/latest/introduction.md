# はじめに

**Box アクティビティ** パッケージには、Box のファイル操作を自動化するためのアクティビティが含まれています。

これらのアクティビティにより、ロボットは次のことが行えます：

* Box 接続スコープを作成する。
* Box ファイルのアップロード、ダウンロード、コピー、移動、削除を行う。
* Box ファイルの詳細情報やファイルロック情報を取得する。

**Box Scope** を使用して認証を行い、Box ファイル用のアクティビティをスコープ内に配置してください。
