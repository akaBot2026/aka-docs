# Manage Access 

## **Overview**

The Manage Access module allows overseeing permissions for a specific entity within Data Service. This comprehensive and detailed permission framework empowers you to seamlessly incorporate all your business users into the service, taking into account their individual proficiency levels and aligning with your specific business needs. Additionally, you have the flexibility to handpick users or agents from your organization and assign relevant permissions to them.

**Note:** By default, organization users have no permission to see anything on Data Service.

Only akaBot Center admins or users with **Admin Permissions** have the authority to grant access permissions and view data on the Data Service.

The following steps enable you to manage access on Data Service:

1. Access the **Data Service**on main the menu
2. Select **Manage Access** tab. The **Manage Access** page is displayed.

The following tabs are available:

* [Assign User](/docs/data-service/latest/user-guide/manage-access/assign-users.md)
* [Assign Agents](/docs/data-service/latest/user-guide/manage-access/assign-agents.md)
* [Assign Tokens](/docs/data-service/latest/user-guide/manage-access/assign-tokens.md)

## **Standard Permissions**

### Admin Permissions

Below is a description of the **Admin Permissions**.

| Permission | Detailed description |
| --- | --- |
| Manage Users and Permission | *Can assign one or more users, agents, tokens to access Data Service * Can edit permissions of other users |

### Entity Permissions

Below is a description of the **Entity Permissions**.

| Permission | Detailed description |
| --- | --- |
| View Entities | Can view list of entities and their fields |
| Customize Entities | Can view, edit, delete entities and their fields |

### Data Permissions

Below is a description of the **Data Permissions**.

| Permission | Detailed description |
| --- | --- |
| Create | Can create data in all or specific entities |
| Read | Can view data in all or specific entities |
| Edit | Can edit and view data in all or specific entities |
| Delete | Can delete data in all or specific entities |
