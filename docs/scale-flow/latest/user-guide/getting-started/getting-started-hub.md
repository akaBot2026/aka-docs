# Getting Started hub

The **Getting Started** page is your friendly checklist after you create an account. It shows what to set up first so customers can message you and your team can reply with confidence.

You do not need to finish everything in one sitting. Complete the steps in any order that fits your business.

---

## Open Getting Started

1. Sign in to ScaleFlow.
2. Open **Getting started** from the user menu / shortcut (or go to the Getting Started page from onboarding links).
3. You will see a greeting such as **Hello, [your name]! Let's get started.** and progress like **2 of 5** completed.

![Open Getting Started](/static/img/open-getting-started.png)

---

## What you will find on the page

### 1) ScaleFlow ecosystem

A simple map of how the main pieces fit together: channels, Inbox, AI, contacts, and integrations.

### 2) Tour guide

Short product tours. You can **View tour**, **Rewatch tour**, or **Do this step** when you want a guided walkthrough of a feature.

### 3) Quick setup wizard

Click **Open quick setup** to launch the guided onboarding wizard (choose a template, teach AI, connect a channel, then confirm). See details below.

### 4) Checklist cards

| Card | What it asks you to do | Where it takes you |
|------|------------------------|--------------------|
| **Connect your channels into one inbox** | Bring customer chats into ScaleFlow | [Channels](../channels/channel-integration) |
| **Set up an AI Agent to share the workload** | Create an AI worker for common questions | [AI Agent](../scaleflow-ai/ai-agent-usage) |
| **Set Up Customer Profile to Unify Contact Records** | Start organizing customer profiles | [Contacts](../operations/contact-management) |
| **Connect a 3rd-Party Integration** | Link tools such as HubSpot or Shopify | [Integrations](../integrations/integration-usage) |
| **Invite Teammates to Collaborate on Chats** | Add colleagues | [User Management](../organization/user-management) |

## Quick setup wizard (optional but recommended)

The wizard is a shorter path for the first day:

1. **Select Template** — for example Zalo Customer Support, Facebook Messenger Support, or Internal Testing & QA
2. **Teach AI** — upload files, crawl a website, or use Google Drive so AI has answers
3. **Connect Channel** — connect the channel that matches your template (or a testing sandbox)
4. **Review & Confirm** — check the summary, then **Complete Setup**

Buttons you will see: **Back**, **Continue**, **Skip**, **Complete Setup**.

Skipping is fine if you prefer to set things up manually from the checklist cards.

![Quick setup wizard](/static/img/wizard.png)

---

## Suggested first-day path

1. Finish [Account Registration](./account-registration) if you have not signed in yet.
2. Open **Getting Started** and run **Open quick setup**, or connect one channel manually.
3. Add a little [Knowledge](../scaleflow-ai/knowledge-usage) (FAQ or website).
4. Create or finish an [AI Agent](../scaleflow-ai/ai-agent-usage).
5. Invite at least one teammate.
6. Send a test customer message and reply in [Inbox](../operations/inbox-usage).
