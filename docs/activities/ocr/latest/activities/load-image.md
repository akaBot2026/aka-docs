# Load Image

RCA.Activities.OCR.LoadImage

## **Description**

Load image from a file into a variable of type System.Drawing.Image.

![load-image](/static/img/load-image.png)

(\*For mandatory)

## **Properties**

**Input**

* **Image Path (String)**\* - Image file path to be loaded.

**Misc**

* **Display Name (String)** - The name of this activity. You can edit the name of the activity to organize and structure your code better.  
  E.g: `[443380046] Load Image`
* **Public (Checkbox)** - Check if you want to public the activity. Remember to consider data security requirement before using this property.

**Output**

* **Output Image (Image)** - Output variable of type System.Drawing.Image.
