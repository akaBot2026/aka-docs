# Introduction

The **Box activities** package contains activities for automating Box file operations.

These activities enable robots to:

* Create a Box connection scope.
* Upload, download, copy, move, and delete Box files.
* Retrieve Box file details and file lock information.

Use **Box Scope** to authenticate and place Box file activities inside the scope.
