## Share an Automation

If you are assigned as a Citizen Developer and you have automated a manual, repetitive task, or process you are encouraged to share your automation project so that other members of your team can beneficiate from using it.

To start the process, please click **Share** -> **Share an Automation**. The automation submission form is displayed. Carefully fill in all the requested information

1. **Overview**: Describe the overview of the idea including Name, Description, Pain points, Negative impact and choose Business Area where the automation can apply
2. **Citizen Developer Assessment**: Fill answers for questions 
3. **Potential Benefit**: During the Automation submission process, an algorithm runs silently to compute the following output scores
